# MongoDB Cheat Sheet 📘

## 📌 Concept
This repository is now arranged as a topic-first learning path.
Large notes were split into focused files so each concept is easier to revise and reuse.

## ✅ Example
Start here by intent:
- Foundations: `foundations/system-databases-and-shell.md`, `foundations/mongosh-built-in-vs-custom-functions.md`
- CRUD flow: `crud/create-and-read.md`, `crud/update-delete-and-upsert.md`
- Querying flow: `queries/projection-logical-and-comparison.md`, `queries/membership-mod-and-range.md`, `queries/null-and-array-operators.md`
- Troubleshooting: `troubleshooting/common-error-patterns.md`

## ⚠️ Common Mistakes
- Keeping all concepts in large single files, which makes revision slower.
- Mixing shell basics, query operators, and troubleshooting in one place.

## 💡 Tips
- Use README as the single entry point and subfolders for details.
- Keep adding new weeks by splitting topic-heavy files instead of appending endlessly.
